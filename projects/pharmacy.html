Pharmacy System

Description

This system is a digital solution for managing pharmacy operations. It handles sales records, customer chats, and back-office processes via database tables and phpMyAdmin. It demonstrates how a pharmacy can modernize its workflows through web technologies and simple CRUD operations.

A platform for pharmacy sales. Keeps records and tracks sales.

Tech used

  • 💻 HTML5, CSS3, JavaScript (Frontend)
  • 🧰 Bootstrap 5 – For responsive layout and clean UI components
  • 🗄️ PHP & MySQL – Backend logic and secure medication database
  • 🔐 Session Handling – For login, role-based access (admin, pharmacist, customer)
  • 📦 CRUD Operations – For managing medicines, orders, and user accounts
  • 📊 Optional: Chart.js – For analytics dashboards
  • 🌐 GitHub for version control & testing
