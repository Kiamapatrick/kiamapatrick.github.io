Point of Sales

Description

The POS system is a robust web application designed to manage products, sales, customers, and suppliers. It includes backend capabilities to view and edit database tables through phpMyAdmin. Built using HTML, CSS, and Python, this project demonstrates practical business-oriented features for inventory and transaction management.

A platform to track sales, products, customers, and suppliers.

Tech Used

  • 💻 HTML, CSS, JavaScript
  • 🧰 Bootstrap (for responsive layout)
  • 📦 LocalStorage (for inventory simulation)
  • ⚙️ Vanilla JS logic for cart, totals, and UI events
