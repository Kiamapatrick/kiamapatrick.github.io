Student Forum Website

Description

This project is an online forum platform tailored for student collaboration. It allows students to create threads, chat in real-time, and view backend data like forum tables and user interactions via phpMyAdmin. It’s a full-stack project showcasing frontend design and backend database integration.

A platform where students can discuss academic topics.

Tech Used

  • 💻 HTML, CSS, JavaScript
  • 📂 PHP – Server-side scripting for authentication and post handling
  • 🗃️ MySQL – Database for user data, posts, and threads
  • 🧰 Bootstrap and Tailwind CSS – For UI consistency and layout
  • 🔒 User Registration/Login – With session handling and access control
  • 📬 Post, Comment, and Reply system – With timestamping
  • 🚀 Deployed using GitHub Pages + backend hosting (e.g., 000WebHost)
