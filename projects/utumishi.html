Description

This project is a tribute to my high school, designed as a fun and informative website that highlights the school’s identity, history, and student life. It includes sections like a homepage, gallery, and contact page, offering a modern and engaging interface. The site was built using HTML, CSS, and Python to demonstrate my growing web development skills during high school.

A fun project for my secondary school.

Tech Used

  • 🖼️ JavaScript (for gallery slideshow)
  • 📨 Formspree or EmailJS
  • 🌐 Deployed with GitHub Pages
