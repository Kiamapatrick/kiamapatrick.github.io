MediKart

Description

MediKart is an online medicine ordering system that uses phpMyAdmin to store and manage medical product data. It includes a user-friendly homepage, chat feature for communication, and backend database access. The project was developed to simulate a simple e-pharmacy, focusing on order handling and database connectivity.

A platform that focuses on phpMyAdmin for ordering medicine.

Tech Used

  • 💻 HTML5, CSS3, JavaScript (Frontend)
  • 🧰 Bootstrap 5 – For responsive layout and clean UI components
  • 🗄️ PHP & MySQL – Backend logic and secure medication database
  • 🔐 Session Handling – For login, role-based access (admin, pharmacist, customer)
  • 📦 CRUD Operations – For managing medicines, orders, and user accounts
  • 📊 Optional: Chart.js – For analytics dashboards
  • 🌐 GitHub for version control & testing
