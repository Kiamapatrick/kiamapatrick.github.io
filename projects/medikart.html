MediKart

Description

MediKart is an online medicine ordering system that uses phpMyAdmin to store and manage medical product data. It includes a user-friendly homepage, chat feature for communication, and backend database access. The project was developed to simulate a simple e-pharmacy, focusing on order handling and database connectivity.

A platform that focuses on phpMyAdmin for ordering medicine.
